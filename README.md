# CigaretteBug — 菸蟲

一款以「收集」和「視覺化」為核心的吸菸記錄 App（iOS + Android）。

靈感來自日本飲酒記錄 App「YoiLog」——掃描菸盒條碼，3D 菸盒在畫面中堆疊滾動，隨時間慢慢變皺變扁，直觀呈現你的吸菸量與品牌偏好。從記錄到戒除，一個 App 涵蓋吸菸者的完整旅程。

## Tech Stack

- Flutter (Dart)
- Firebase (Firestore / Auth / FCM)
- 3D: three_dart + forge2d
- iOS Widget: Swift WidgetKit
- Android Widget: Kotlin Glance
